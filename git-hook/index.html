Git 钩子使用

2023-12-01
类别: 软件技术
标签: git

Git 里的钩子 hook 就是在 git 的运行流程中，在指定的节点上会自动执行的脚本。

钩子默认是放在 git 仓库路径下的 $GIT_DIR/hooks 里，每个钩子都有固定的名字，并要加上可执行权限。
